# 养老金计算器 - 需求文档

## 1. 用户管理

### 1.1 用户配置
- 出生年份
- 性别（男/女）
- 配置保存到本地 IndexedDB

### 1.2 健康档案
- 身高、体重、BMI 计算
- 吸烟/饮酒习惯
- 运动频率
- 慢性病情况
- 预期寿命计算（基于 WHO 数据和生活方式调整）

## 2. 养老金配置

### 2.1 基本配置
- 养老金类型：基本养老金 / 补充养老金 / 两者都有
- 当前个人账户余额
- 退休年龄
- 是否有过渡性养老金（中人）
- 视同缴费年限
- 视同缴费指数
- 过渡性养老金计发系数

### 2.2 缴费阶段
- 支持多阶段配置
- 每阶段包含：
  - 开始年份、结束年份
  - 月缴费基数
  - 社平工资（用于计算缴费指数）
  - 个人缴费比例
  - 单位缴费比例
  - 每年缴费月数
  - 备注
- 支持"灵活就业"类型阶段：
  - 可选缴费档次（60%/80%/100%/200%/300%）
  - 每年缴费月数（1-12 个月）
  - 自动根据社平工资计算缴费基数
- 支持在职阶段"结束后自动转灵活就业"

## 3. 缴费记录管理

### 3.1 手动添加记录
- 年份
- 月缴费基数
- 社平工资
- 个人缴费比例
- 单位缴费比例
- 缴费月数

### 3.2 浙里办 PDF 导入
- 解析浙里办参保证明 PDF
- 自动提取：
  - 参保年月
  - 缴费基数
  - 个人缴费额
  - 单位缴费额
  - 缴费月数
- 支持多行记录合并（同一月份多段缴费）
- 支持多页 PDF
- 导入预览和确认

### 3.3 记录展示
- 按年份排序
- 显示缴费基数、社平工资、缴费指数、缴费月数、个人缴费额
- 支持编辑和删除

## 4. 社平工资管理

### 4.1 数据导入
- 支持 JSON 格式导入
- 包含：年份、月社平工资、年社平工资
- 支持浙江省加权平均工资和杭州市非私营单位数据

### 4.2 AI 提示词生成
- 根据导入的数据自动生成提示词
- 提示词用于向 AI 询问社平工资数据来源

### 4.3 数据展示
- 表格展示历年社平工资
- 显示增长率

## 5. 养老金计算

### 5.1 计算公式
- **基础养老金** = 退休时上年度社平工资 × (1 + 平均缴费指数) / 2 × 总缴费年限 × 1%
- **个人账户养老金** = 退休时个人账户余额 / 计发月数
- **过渡性养老金** = 退休时上年度社平工资 × 视同缴费指数 × 视同缴费年限 × 计发系数
- **月养老金总额** = 基础养老金 + 个人账户养老金 + 过渡性养老金

### 5.2 计发月数
- 根据退休年龄查表（国发〔2005〕38 号）
- 40 岁：233 个月 ~ 70 岁：56 个月

### 5.3 个人账户记账利率
- 2010-2015：2.25%-3.25%（银行定期存款利率）
- 2016-2026：人社部公布值（3.5%-8.31%）
- 其他年份：默认 3%

### 5.4 社平工资推算
- 使用导入的社平工资数据计算复合年增长率（CAGR）
- 推算退休时的社平工资
- 公式：退休时社平工资 = 最新已知社平工资 × (1 + CAGR)^(退休年份 - 最新已知年份)

### 5.5 缴费指数计算
- 每年缴费指数 = 缴费基数 / 当年社平工资
- 平均缴费指数 = Σ(缴费指数 × 缴费月数) / 总缴费月数
- 优先使用导入的社平工资数据，未导入时使用记录中的值

### 5.6 个人账户累积
- 历年明细从 0 开始逐年累加
- 当年缴费年底计入，然后计算年度利息
- 用户填写的当前余额替代 PDF 导入部分的推算值
- 灵活就业阶段的缴费和利息单独累积

## 6. 测算结果展示

### 6.1 主要结果
- 月养老金
- 年养老金
- 养老金总额（到预期寿命）
- 可领取年数

### 6.2 购买力折算
- 折算现在购买力 = 退休时月养老金 × (现在社平工资 / 退休时社平工资)
- 相当于现在月薪的百分比
- 显示现在月薪具体金额

### 6.3 分项展示
- 基础养老金
- 个人账户养老金
- 过渡性养老金
- 退休时个人账户余额
- 退休时社平工资
- 平均缴费指数
- 总缴费年限
- 计发月数

### 6.4 历年缴费明细
- 逐年展示：年份、缴费基数、社平工资、缴费指数、当年缴入、个人账户余额

## 7. 灵活就业方案对比

### 7.1 默认对比方案
- 最低档缴满（60%，到退休）
- 100%档次缴满
- 最低档缴 5 年
- 提前 3 年退休
- 最高档缴满（300%）

### 7.2 自定义方案
- 缴费档次：60/80/100/200/300%
- 缴费年限：1-30 年
- 退休年龄：50-70 岁
- 每年缴费月数：1-12 个月

### 7.3 对比维度
- 个人总缴费
- 月养老金
- 折算到现在购买力的月养老金
- 相当于现在月薪的百分比
- 养老金总额
- 投入产出比（养老金总额 / 个人总缴费）

### 7.4 方案选用
- 点击"选用"按钮选中方案
- 选中的方案用于养老金测算计算
- 养老金页面显示当前选用方案
- 可取消选用

### 7.5 计算过程展示
- 退休时社平工资
- 社平工资增长率
- 平均缴费指数
- 总缴费年限
- 基础养老金公式
- 个人账户养老金公式

## 8. 充足性分析

### 8.1 退休资产充足率
- 计算退休资产 / 退休资金缺口
- 显示充足率百分比

### 8.2 资金缺口
- 基于养老金和预期支出计算

## 9. 数据持久化

### 9.1 本地存储
- 使用 IndexedDB 存储所有数据
- 支持离线使用

### 9.2 数据导出
- 支持导出为文件

## 10. 技术栈

- Vue 3 + TypeScript + Vite
- Pinia 状态管理
- PWA 支持（离线使用）
- pdfjs-dist 解析 PDF
- Dexie.js 操作 IndexedDB
